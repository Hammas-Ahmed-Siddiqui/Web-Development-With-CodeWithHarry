Description:
This HTML file, named "index.html," represents a simple web page. It contains the following elements and attributes:

1. <!DOCTYPE html>: This declaration defines the document type and version of HTML being used (HTML5).

2. <html lang="en">: The root element of the HTML document. The "lang" attribute is set to "en" to indicate that the document is in English.

3. <head>: This section contains meta-information about the document, such as character encoding and viewport settings.

   - <meta charset="UTF-8">: Specifies that the character encoding for the document is UTF-8, which supports a wide range of characters.
   
   - <meta name="viewport" content="width=device-width, initial-scale=1.0">: Sets the viewport configuration for responsive web design, ensuring the page adapts to the device's width and initial scale.

   - <title>Document</title>: Defines the title of the web page, which is displayed in the browser's title bar or tab.

4. <body>: The main content of the web page is enclosed within the <body> tags.

   - <video src="Bridge.mp4" controls></video>: A <video> element is embedded in the page. It displays a video file named "Bridge.mp4." The "controls" attribute adds playback controls (play, pause, volume, etc.) to the video player, allowing users to interact with the video.

This HTML file, when opened in a web browser, will display a video player with controls, allowing users to play the "Bridge.mp4" video file.
